sun4u-ac

Provided by
sunfire-devices
Class Hierarchy
conf-object → log-object → sun4u-ac
Interfaces Implemented
log_object, io_memory
Description
The sun4u-ac device implements the address controller found on all SunFire system boards. It currently only contains dummy registers for reading and writing. All use of unimplemented functionality in the sun4u-ac is reported.
