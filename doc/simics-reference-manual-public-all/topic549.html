Core_Back_To_Front

Provided By
Simics Core
Callback Type
void (*)(lang_void *callback_data,
conf_object_t *trigger_obj);
Description
Called when Simics returns to the frontend command loop. This happens whenever Simics has processed the previous command from the command line and is about to process the next one.
