MMU_Data_TLB_Replace

Provided By
cheetah+mmu, panther-mmu, cheetah-mmu, jaguar-mmu, spitfire-mmu, jalapeno-mmu
Callback Type
void (*)(lang_void *callback_data,
conf_object_t *trigger_obj, integer_t tag_reg,
integer_t data_reg);
Index
context_number
Description
Triggered when an entry in the Cheetah+ MMU is removed from the D-TLB due to replacement. A break request will stop the simulation on the next instruction.
