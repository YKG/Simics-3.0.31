file-cdrom

Provided by
file-cdrom
Class Hierarchy
conf-object → log-object → file-cdrom
Interfaces Implemented
log_object
Description
This extension allows a file in ISO9660 format to be accessible from the simulated target machine. This is a convenient way to import files into Simics.
