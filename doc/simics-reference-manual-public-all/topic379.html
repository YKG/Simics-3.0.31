spitfire-mmu

Provided by
spitfire-mmu
Class Hierarchy
conf-object → log-object → spitfire-mmu
Interfaces Implemented
log_object, mmu
Description
The spitfire-mmu models the MMU in UltraSPARC-I (Spitfire) and UltraSPARC-II (Blackbird) processors.
