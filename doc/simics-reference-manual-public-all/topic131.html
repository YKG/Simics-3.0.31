io-apic

Provided by
io-apic
Class Hierarchy
conf-object → log-object → io-apic
Interfaces Implemented
log_object, io_memory
Description
The IOAPIC device implements the functionality of the I/O-APIC (82093AA). The IOAPIC is connected to an APIC-bus.
