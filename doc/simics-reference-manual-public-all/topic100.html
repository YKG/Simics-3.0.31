GT64120-pci

Provided by
GT64120
Class Hierarchy
conf-object → log-object → GT64120-pci
Interfaces Implemented
log_object, bridge, pci_bridge, pci_interrupt, io_memory, pci_device
Description
The GT64120-pci class models the the host to PCI bridges of the GT64120. Currently only 32-bit operation is supported.
