sun4u-systemregs

Provided by
sunfire-devices
Class Hierarchy
conf-object → log-object → sun4u-systemregs
Interfaces Implemented
log_object, io_memory
Description
The sun4u-systemregs device implements the System Control Registers found on the clock board. Most of the registers are dummy, although they do return valid data on reads, and check for valid data on writes.
