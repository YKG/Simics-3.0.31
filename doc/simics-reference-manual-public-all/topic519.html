pci_interrupt

Implemented By
GT64120-pci, LX164-IRQ-PLD, MD1535D-isa, artesyn-pm-ppc, i82461GX, piix4_isa, ppc405gp-pci, ppc405gp-uic, ppc440gp-pci, ppc440gp-uic, ppc440gx-pci, ppc440gx-uic, uPD66566S1016, wire
Description
This interface should only be used when a device other than the bridge handles PCI interrupts on the PCI bus. The initiating device is specified with a PCI device number, and the pin represents PCI interrupt lines (A, B, C, or D) as numbers in the range of 0 to 3. 
typedef struct {
        void (*raise_interrupt)(conf_object_t *obj, conf_object_t *irq_obj,
                                int device, int pin);
        void (*lower_interrupt)(conf_object_t *obj, conf_object_t *irq_obj,
                                int device, int pin);
} pci_interrupt_interface_t;
