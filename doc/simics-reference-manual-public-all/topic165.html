microwire-eeprom

Provided by
microwire-eeprom
Class Hierarchy
conf-object → log-object → microwire-eeprom
Interfaces Implemented
log_object, microwire
Description
This device models a generic, 3-wire (microwire) serial EEPROM (for example 93Cxx).
