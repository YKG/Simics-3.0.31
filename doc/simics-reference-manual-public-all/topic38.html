cassini_sub

Provided by
cassini
Class Hierarchy
conf-object → log-object → cassini_sub
Interfaces Implemented
log_object, io_memory, pci_device
Description
The cassini_sub class implements the PCI functions 1, 2 and 3 of a Cassini GBit Ethernet device. Three objects of this class is required for each cassini object to form a complete device.
