malta-system

Provided by
malta-components
Class Hierarchy
conf-object → log-object → component → top-component → malta-system
Interfaces Implemented
log_object, component
Description
The "malta-system" component represents a MIPS Malta development board with a MIPS 4Kc or 5Kc processor and a Galileo GT64120 system controller.
